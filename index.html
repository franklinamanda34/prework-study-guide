Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML defines the structure of your content
  • HTML is short for Hypertext Markup Language

CSS

  • CSS is an acronym used for Cascading Style Sheets
  • CSS defines how content is presented to users
  • a margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • Git is a version control system that allows you to stay up to date and keep track of each version of your project
  • Github provides an easy to access document containing important and commonly known Git commands
  • Git is a version control software whereas github is a cloud based service where your project and repositories will live
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it

javascript

  • javascript is generally used in combination with HTML and CSS
  • JS is a critical component in making web pages as dynamic as possible for users
  • JS powers the ease of use for users when visting a webpage
